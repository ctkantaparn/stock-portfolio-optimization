# Stock Portfolio Optimization

## Overview
This project analyzes historical stock data, calculates returns, volatility, correlations, and optimizes a portfolio using Sharpe ratio. Visualizations include price trends, return distributions, and risk-return tradeoffs.

## Folder Structure
- `data/` - Raw or processed stock data
- `notebooks/` - Jupyter notebook for analysis
- `src/` - Python modules for data loading, analytics, and optimization
- `visuals/charts/` - Generated charts
- `summary/` - Executive summary (PDF)
- `requirements.txt` - Python dependencies

## Usage
```bash
pip install -r requirements.txt
jupyter notebook notebooks/portfolio_analysis.ipynb
